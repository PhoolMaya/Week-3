Bootstrap

Components of bootstrap

Buttons: Bootstrap has different styles for buttons, they are;primary, secondary, success, danger, and more. Buttons cab be easily created with different sizes and shapes.

Forms and Input Groups: Bootstrap provides styling for form elements like input fields, checkboxes, radio buttons, and dropdowns. Input groups allow you to combine input fields with buttons or other elements.

Navigation Components: Bootstrap includes components like the navbar, tabs, and paging. The navbar helps create responsive navigation menus, while tabs allow you to organize contents.

Alerts: Alerts are used to display messages to users. Bootstrap provides different alert styles (success, warning, info, danger) that can be customized.

Progress Bars: Bootstrap offers progress bars to indicate the completion status of a task or process.

Typography: Bootstrap provides consistent typography styles for headings, paragraphs, and other text elements.
